# 技术路线图 (Roadmap) - Phase 3: 效率进阶与桌面原生化

**当前状态**: Phase 3 核心功能已完成 (桌面原生特性已上线)

## ✅ 已完成回顾 (Phase 1, 2 & 3 早期)

- [x] **基础设施**: Electron 跨端架构、Vercel API 路由优化、离线/在线自适应寻址。
- [x] **核心功能**: 跨端登录同步、日志/待办 CRUD、AI 周报生成。
- [x] **桌面原生特性 (NEW)**:
  - [x] **全局秒开快捷键**: 支持用户在设置页自定义呼出快捷键（默认 `Alt+Space`），即时生效。
  - [x] **系统托盘集成**: 状态指示、快捷菜单（置顶、显示、退出）。
  - [x] **窗口置顶模式**: Header 增加 Pin 图标，支持 UI 级与同步菜单控制。

  - [x] **多维洞察看板**: 引入 Radar 领域平衡图、趋势柱状图。
  - [x] **AI 智能标签**: 输入框集成“AI 标签”按钮，自动分析内容并提取 #标签。
- [x] **导出增强 (NEW)**:
  - [x] **Markdown 下载**: 支持周报一键下载为 `.md` 文件，完美适配 NotebookLM。

## 🚀 Phase 3: 深度优化与自动化 (下半程)

**核心目标**: 强化自动化流程，提升长期记录的复盘价值。

### 3.4 自动化与智能提醒 (Automation)

- [ ] **每日定时汇报 (Daily Push)**:
  - 每天 18:00 自动触发聚合，并向用户发送桌面气泡通知。
- [ ] **遗忘提醒**:
  - 如果连续 2 天没有活跃，桌面端通过托盘图标闪烁或通知提醒“捕捉一下今日灵感”。

### 3.5 视觉与交互打磨 (UX Polish)

- [ ] **窗口自适应弹性布局**:
  - 针对桌面端小窗模式（类似 Spotlight 搜索框）优化 UI 紧凑度。
- [ ] **微交互动效**:
  - 切换 Tab 时的平滑平移，以及“快速捕捉”后的流星飞入效果。

### 3.6 互联生态 (Ecosystem)

- [ ] **多格式导出模板**:
  - 增加 Notion 风格模板与 Obsidian 双向链接兼容。
