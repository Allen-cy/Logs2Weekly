# AI Productivity Hub - 产品需求文档 (PRD) v2.0

## 1. 文档概览

- **目标**：建立完整的用户体系，实现多设备配置同步、数据隔离及高效检索。

---

## 2. 核心功能描述

### 2.1 用户认证系统

- **注册/登录**：支持“手机号/邮箱 + 密码”多渠道认证。
- **安全性**：采用 `bcrypt` 强哈希加密。

### 2.2 用户配置云同步

- **自动保存**：AI 供应商设置及 API Key 自动同步。
- **跨端加载**：登录后自动恢复最近使用的模型配置。

### 2.3 现代化日志搜索

- **全文检索**：Dashboard 支持对内容及标签的实时模糊匹配。
- **性能支撑**：基于 `pg_trgm` 的三元组索引优化。

---

## 3. 技术架构

- **前端**：React + TypeScript + Tailwind。
- **后端**：FastAPI + Pydantic。
- **数据库**：Supabase (PostgreSQL) + pg_trgm。
