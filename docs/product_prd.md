# AI Productivity Hub - 产品需求文档 (PRD) v3.2.0

## 1. 文档概览

- **目标**：建立完整的用户体系，实现多设备配置同步、桌面端原生深度集成、可视化洞察及周报历史归档。

---

## 2. 核心功能描述

### 2.1 用户认证系统

- **注册/登录**：支持“手机号/邮箱 + 密码”及 GitHub OAuth 认证。
- **安全性**：采用 `bcrypt` 强哈希加密。

### 2.2 用户配置云同步

- **自动保存**：AI 供应商设置及 API Key 自动同步。
- **原生配置同步 (NEW)**：支持全局快捷键、留存时长等桌面端偏好设置云端备份。

### 2.3 现代化日志搜索

- **全文检索**：Dashboard 支持对内容及标签的实时模糊匹配。

### 2.4 无压快记与 AI 自动聚合

- **聚合机制**：系统可手动或定时（每晚 18:00）通过 AI 聚类零散内容，生成日报。

### 2.5 数据洞察与周报归档 (Phase 3)

- **可视化看板**：接入 `recharts`，展示标签分布比例（PieChart）、7 天产出活力趋势（BarChart）及领域平衡雷达图（RadarChart）。
- **周报历史归档**：支持将生成的 AI 周报一键保存为快照，持久化存储。
- **高级导出**：提供周报 Markdown 格式下载，适配 NotebookLM。

### 2.6 全局搜索与待办事项专项

- **全局搜索重构**：搜索框全应用一致的搜索体验。
- **仿苹果待办模块**：提供独立的“待办事项”视图。

### 2.7 待办事项四象限看板

- **艾森豪威尔矩阵**：新增四象限视图（Quadrant View），根据“重要/紧急”维度自动分类。

### 2.8 桌面端原生集成 (Desktop Native Integration) - NEW

- **全局系统热键**:
  - 支持 `Alt+Space`（默认）一键呼出/隐藏应用。
  - **自定义配置**: 用户可在设置界面自定义快捷键组合，即时重注册生效。
- **系统托盘 (System Tray)**:
  - 常驻菜单栏，支持快捷操作菜单。
  - 窗口状态联动（显示/隐藏/置顶）。
- **窗口置顶 (Always on Top)**:
  - UI 级 Pin 功能，支持应用窗口始终浮于最前。
- **检查更新系统**:
  - 版本追踪与手动查更按钮，同步展示版本演进史及下载地址。

---

## 3. 技术架构

- **前端/桌面端**：React + TypeScript + Tailwind + Recharts + Electron。
- **后端**：FastAPI + Pydantic + Supabase SDK。
- **数据库**：Supabase (PostgreSQL) + pg_trgm。
