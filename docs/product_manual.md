# AI Productivity Hub 用户手册 (v3.0)

欢迎使用 AI Productivity Hub！这是一个极简、高效的生产力记录与周报生成工具。

## 1. 快速开始 (Quick Start)

1. **注册/登录**: 使用手机号或邮箱注册，登录后即可开始记录。系统支持自动保持登录状态，无需每次重复输入。
2. **记录日志**: 在主界面输入框记录此时此刻的想法、工作或任务。
   - 输入 `- [ ] 任务` 可创建待办事项。
   - 直接输入文字则默认为“笔记”。
3. **生成周报**: 点击右上角 **"生成周报"** 按钮，AI 将根据本周记录自动汇总出结构化报告。

## 2. 核心功能 (Core Features)

### A. 智能记录 (Smart Logging)

- **快捷输入**: 支持 Markdown 语法快速标记任务状态。
- **状态切换**: 点击任务前的复选框，可快速切换 `进行中` / `已完成`。
- **打标签**: 记录时可添加 `#标签` (如 `#工作`, `#学习`)，系统会自动统计并展示在 "Top Contexts"。

### B. 仪表盘 (Dashboard)

- **动态问候**: 顶部 Header 会根据当前时间 (早/中/晚) 给予贴心问候。
- **活跃度热力图**: “近期活跃度”组件展示您近 30 天的记录频率，颜色越深代表记录越多。
- **搜索日志**: 使用搜索框或点击搜索图标，支持按关键词快速查找历史记录。
- **标签统计**: "Top Contexts" 展示您最常关注的领域。

### C. 周报生成 (Weekly Report)

- **多模型支持**: 支持切换 Gemini 1.5/2.0 或 Kimi 等 AI 模型。
- **结构化输出**: 生成的周报包含：
  - **Executive Summary**: 核心成就概览。
  - **Focus Areas**: 时间精力分配比重。
  - **Pulse Stats**: 任务完成数与深度工作时长。
  - **Highlights**: 本周高光时刻。

## 3. 设置与其他 (Settings & More)

- **INSIGHTS (数据洞察)**: 透视你的生产力规律。通过饼图查看各领域的精力分配，并在趋势图中观察最近 7 天的产出活力。
- **ARCHIVE (周报归档)**: 这里是你所有过往成就的博物馆。您可以回溯任何一份曾经保存过的周报。
- **保存与导出**: 在生成周报后，您可以点击右上角的 **"保存到历史"** 进行归档，或点击 **"复制 Markdown"** 快速同步到笔记软件。
- **API 配置 / 个人资料**: 点击导航栏中的 **"SETTINGS"** 即可进入。
  - 如果您尚未配置 API Key，将进入快速引导流。
  - 如果已配置，将进入个人中心，您可以修改昵称、密码或重新调整 AI 引擎。
- **使用指南**: 点击右上角的 **"GUIDE"** 按钮，随时查看详细的操作说明。
