# AI Productivity Hub - 用户手册

## 🚀 快速上手

### 1. 账号管理

- **注册**：提供手机号、邮箱即可创建。
- **登录**：支持手机号或邮箱登录。

### 2. 配置 AI

- 在 **“设置 (Setup)”** 页面输入供应商和 API Key。
- 点击“测试并保存”后，系统会记住您的偏好。

### 3. 日志搜索

- 使用 **“工作台 (Dashboard)”** 顶部的检索框实时筛选历史。

### 4. 无压快记与日报

- **记录**：直接在“收纳盒 (Inbox)”输入内容，无需担心日期。
- **日报**：每天下午 18:00，AI 会为您自动整理收纳盒中的内容，生成一份聚合日报，可在看板中查看。

---

## ❓ 常见问题

- **Q: 换了电脑还要重输 Key 吗？**
  - A: 不需要，登录账号后自动同步。
- **Q: 为什么搜不到内容？**
  - A: 建议记录时多加 `#标签`。
