# 技术路线图 (Roadmap) - Phase 3 & Beyond

**当前状态**: Phase 2 已完成 (UI 焕新 + 生产环境稳定)

## ✅ 已完成 (Phase 1 & 2)

- [x] **核心功能**: 日志增删改查、周报生成、多模型支持。
- [x] **UI/UX 升级**: 动态问候、热力图、暗黑模式优化、响应式布局。
- [x] **基础设施**: Vercel Serverless 部署、跨域代理配置、依赖管理。
- [x] **体验优化**: 登录持久化、智能搜索、用户指南集成。

## 🚀 Phase 3: 数据洞察与高级功能 (Current Focus)

**目标**: 让数据“活”起来，提供更深度的分析与更便捷的导出。

### 3.1 数据可视化增强

- [ ] **周报历史归档**: 支持查看和对比往期周报。
- [ ] **趋势图表**: 引入 Chart.js/Recharts 展示任务完成率趋势。
- [ ] **标签分类统计**: 饼图展示各领域精力分配。

### 3.2 高级导出功能

- [ ] **Markdown 导出**: 一键复制周报为 Markdown 格式 (适配 Notion/Obsidian)。
- [ ] **PDF/图片导出**: 生成精美的周报分享卡片。

## 🔮 Phase 4: 移动端与多端同步 (Next Steps)

- [ ] **PWA 支持**: 支持添加到手机主屏幕，离线记录。
- [ ] **微信/飞书集成**: 通过聊天窗口快速发送日志 (需后端 API 支持)。

## 🛠️ 技术债 (Tech Debt)

- [ ] **Auth 升级**: 引入 JWT 或 Supabase Auth 替代简单的 ID 验证。
- [ ] **测试覆盖**: 补充关键路径的 E2E 测试 (Cypress/Playwright)。
